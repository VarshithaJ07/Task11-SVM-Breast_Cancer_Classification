# Task11-SVM-Breast_Cancer_Classification


This task implements Support Vector Machine (SVM) for classifying breast cancer
as malignant or benign using the sklearn breast cancer dataset.

## Dataset
- sklearn.datasets.load_breast_cancer

## Steps Performed
- Data loading and exploration
- Feature scaling using StandardScaler
- Train-test split with stratification
- Linear and RBF kernel SVM comparison
- Hyperparameter tuning using GridSearchCV
- Model evaluation using accuracy, confusion matrix, ROC curve, and AUC
- Saving trained model using joblib

## Tools & Libraries
- Python
- Scikit-learn
- NumPy
- Matplotlib
- Seaborn
